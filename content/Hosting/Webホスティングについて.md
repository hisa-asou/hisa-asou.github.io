## GitHub Pages

* やってみた [github](https://hisa-asou.github.io/)
* プロジェクト名が固定(_username_.github.io)なのでGitHubアカウントで1個のサイトしか使えない。
* Markdown のソースを編集して、HTMLをbuildして commit/push するだけなので個人作業としては簡単
* まあ認証をどうするかとかあるから難しい気はします。
